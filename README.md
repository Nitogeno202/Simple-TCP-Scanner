# Simple TCP Port Scanner  
Este es un escáner de puertos TCP básico desarrollado en Python como parte de mi aprendizaje en ciberseguridad.

## 🛠️ Características  
- Escaneo de puertos comunes (22, 80, 443, etc.).  
- Implementación de \`socket\` para conexiones TCP.  
- Manejo de Timeouts para optimizar la velocidad en Linux.  
- Feedback visual en tiempo real.

## 🚀 Cómo usar  
1. Clonar el repositorio.  
2. Ejecutar con: \`python3 scanner.py\`  
3. Ingresar la IP del objetivo.
